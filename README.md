# 🪑 教室座位智能编排系统 (SeatWise)

一个功能强大的教室座位智能编排系统，支持概率公平抽取、多维成绩热力图、智能推荐、拖拽互换等功能。

## ✨ 核心功能

### 🎲 智能抽取
- **公平可预测模式**：基于权重的公平概率算法
- **公平不可预测模式**：带随机扰动的抽取
- **性别均衡 / 防扎堆**：自动保持性别分布均衡，防止过度集中
- **一键自动抽取**：支持连续自动排座

### 🧠 智能推荐
- **多维评估**：学业成绩、性格互补、爱好搭配、职务平衡、性别均衡
- **互换建议**：基于综合评分推荐座位互换
- **自定义算法**：支持导入自定义评估算法

### 📊 数据可视化
- **成绩热力图**：多维度热力图（综合/单科/平均）
- **概率预测面板**：实时显示每个座位的抽取概率
- **蒙特卡洛模拟**：预演多次排座结果，分析座位分布

### 🖱️ 交互操作
- **拖拽排座** / **点击互换** / **右键菜单**
- **Ctrl+K 命令面板**：快速搜索学生和操作

### 📤 导入导出
- **Excel 导入** (.xlsx) / **截图导出** (PNG/JPEG)
- **高级导出**：模块化 JSON 导出/导入
- **打印支持**：优化的打印布局

### 🔌 插件系统 & 🎨 主题
- 插件注册、安全沙箱、模块热替换
- 4套主题 + 8种强调色 + 深色模式

## 🚀 快速开始

1. 在浏览器中打开 `index.html`
2. 通过 Excel 导入学生名单（或手动添加）
3. 设置教室布局（排数 × 列数）
4. 点击「抽取下一个」开始排座
5. 使用拖拽或互换调整座位

## 🛠️ 技术栈

- **前端**：原生 HTML / CSS / JavaScript（无框架依赖）
- **外部库**：[xlsx](https://github.com/SheetJS/sheetjs) / [html2canvas](https://github.com/niklasvh/html2canvas) / [pinyin-pro](https://github.com/zh-lx/pinyin-pro)
- **设计风格**：Apple 风格毛玻璃 UI，支持深色模式

## 📝 学生名单 Excel 格式

| 姓名 | 性别 | 午休 | 语文 | 数学 | 英格 | ... |
|------|------|------|------|------|------|-----|
| 张三 | 男   | 是   | 85   | 92   | 78   | ... |

- **必填**：姓名
- **可选**：性别（男/女）、午休（是/否）、各科成绩、性格、爱好、职务

## 🎯 使用技巧

1. **Ctrl+K** 打开命令面板
2. **右键点击座位** 查看上下文菜单
3. **拖拽学生** 从待选区到座位
4. **点击列头** 禁用整列（过道）
5. **热力图** 可视化成绩分布
6. **蒙特卡洛模拟** 预演排座结果
